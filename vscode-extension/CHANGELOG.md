# Changelog

All notable changes to the TaskOS extension will be documented in this file.

## [0.1.0] - 2026-02-17

### Added

- Task management sidebar with tree view
- View and manage TaskOS tasks directly in VS Code / Cursor
- AI code generation from task descriptions
- Create tasks from selected code or text
- Send tasks to AI Agent for automated implementation
- Create Pull Requests directly from VS Code
- Real-time task sync with configurable auto-refresh interval
- Right-click context menu integration
- API key configuration via command palette
