# Changelog

All notable changes to the TaskOS extension will be documented in this file.

## [0.1.1] - 2026-02-17

### Added

- **AI Agent Pipeline** — full workflow: Plan → Implement → Test → Review → Autofix → Open PR
- **Self Code Review** — AI reviews every PR for security, performance, and best practices
- **Custom Profiles** — define Code Style and Code Review profiles to enforce team standards
- **Automatic Testing** — detects and runs tests, with autofix for failures
- **Rich PR Output** — PRs include structured summaries, review findings, and test results
- **Profile Presets** — Default and Strict presets for quick setup
- **Pipeline Command** — `TaskOS: Run Pipeline & Create PR` via command palette

### Changed

- Redesigned main panel with modern, professional UI
- Redesigned API key connection panel
- Updated extension description and metadata

## [0.1.0] - 2026-02-17

### Added

- Task management sidebar with tree view
- View and manage TaskOS tasks directly in VS Code / Cursor
- AI code generation from task descriptions
- Create tasks from selected code or text
- Send tasks to AI Agent for automated implementation
- Create Pull Requests directly from VS Code
- Real-time task sync with configurable auto-refresh interval
- Right-click context menu integration
- API key configuration via command palette
